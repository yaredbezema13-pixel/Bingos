# 🎉 Bingo Game

A simple Bingo number drawing game built with **HTML, CSS, and JavaScript**.

---

## 📌 Features
- 🎲 Random number generator (1–75)
- 🔤 B I N G O letter system
- 🚫 No duplicate numbers
- 🏁 Game over when all numbers are used
- 🎨 Clean and simple UI
- 📁 Separate files (HTML, CSS, JS)

---

## 📁 Project Structure---

## 🚀 How to Run
1. Download or clone the project
2. Open `index.html` in your browser
3. Click **DRAW NUMBER** to start the game

---

## ⚙️ How It Works
- Click the button to generate a random number
- Each number belongs to a letter group:
  - B = 1–15
  - I = 16–30
  - N = 31–45
  - G = 46–60
  - O = 61–75
- Numbers never repeat
- Game ends after 75 numbers
